# 🔌 ConnectionDC

Plugin central de integração com Discord para servidores Minecraft (Paper).

## 🚀 Sobre

O **ConnectionDC** é um plugin que centraliza a conexão com o Discord usando JDA, permitindo que outros plugins enviem mensagens e embeds sem precisar criar múltiplas conexões.

---

## 📦 Funcionalidades

- 🔗 Conexão única com o bot do Discord
- 📤 Envio de mensagens simples
- 🎨 Envio de embeds
- ⚡ API simples para outros plugins
- 🧠 Evita conflitos de JDA (classloader)

---

## ⚙️ Configuração

Arquivo `config.yml`:

```yml
discord:
  token: "SEU_TOKEN_DO_BOT"

```
